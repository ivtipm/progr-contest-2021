Олимпиада по программированию 2021 в ЗабГУ

10 апреля, 13:00 - 14:30

корпус энергетического факультета, аудитория 401 (4 этаж)

Условия

  • Участвуют все студенты ЗабГУ
  • Будут призы за 1, 2 и 3 место: сертификаты в сеть магазинов ДНС
  • Будут сертификаты участникам и дипломы победителям
  • Языки программирования: Python, С++, С, Java, С#, Pascal (Lazarus)
  • Готовое решение для каждой задачи обязательно должно содержать исходный код программы и все файлы необходимые для компиляции (если требуется) и запуска
  • В основном файле исходного кода должно обязательно содержатся полные ФИО автора и название группы
  • Работы на проверку принимаются в течении всего времени проведения олимпиады + 10 минут
  • Если в решении будет обнаружен плагиат, то решение аннулируется
  • Организаторы олимпиады оставляют за собой право дисквалифицировать участника за нарушение правил
  • Победители (1, 2 и 3 место) будут объявлены в течении недели после окончания олимпиады

Задания

Задание 1 Задание 2 Задание 3
  • Выполняйте задание на одном из рекомендованных языков программирования
  • Сохраните каждое задание в отдельной папке, с номером задания: 1, 2, 3
  • Сохраните эти папки в папку с вашей фамилией и инициалами, например 'Иванов И.И.'
  • В заданиях 1-2 не используйте стандартные алгоритмы из библиотеки языка программирования: поиск элемента в массиве, сортировка и т.д. Реализовывайте эти алгоритмы самостоятельно
  • В задании 3 рекомендуется использовать Python и можно использовать любые популярные библиотеки для любого язка программирования.
