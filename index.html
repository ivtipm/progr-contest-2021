Олимпиада по программированию 2020 в ЗабГУ

11 апреля, 11:00 - 14:00

Чтобы не содействовать распространению COVID-19 олимпиада пройдёт в заочной форме.

Условия

  • Участвуют все студенты ЗабГУ
  • Будут призы
  • Языки программирования: Python, С++, С, Java, С#, Pascal (Lazarus)
  • Для отправки заданий на проверку будет доступен адрес электронной почты и\или репозиторий на github
  • Готовое решение для каждой задачи обязательно должно содержать исходный код программы и все файлы необходимые для компиляции (если требуется) и запуска
  • В основном файле исходного кода должно обязательно содержатся полные ФИО автора и название группы
  • Работы на проверку принимаются в течении всего времени проведения олимпиады + 10 минут
  • Если в решении будет обнаружен плагиат, то решение аннулируется
  • Организаторы олимпиады оставляют за собой право дисквалифицировать участника за нарушение правил
  • Победители (1, 2 и 3 место) будут объявлены в течении недели после окончания олипиады

Задания

Будут опубликованы здесь 11 апреля в 11:00.
